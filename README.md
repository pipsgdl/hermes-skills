# Hermes Skills Global

Skills compartidos entre las 3 instancias Hermes Agent de Felipe:
- **Mac Mini** (Hermes Obsidian local)
- **Rogger** (Hermes Zack corporativo)
- **Ozzy** (Hermes Mr Crowley personal)

Cada equipo monta este repo como `skills.external_dirs` en `~/.hermes/config.yaml`.
Cron pull cada 30 min para sincronización.
